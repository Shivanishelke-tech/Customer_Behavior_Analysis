# Customer_Behavior_Analysis
Project Overview

This project focuses on analyzing customer shopping behavior using Python, SQL Server, and Power BI. The goal of the project is to identify customer trends, purchasing patterns, revenue insights, and engagement behavior through data analysis and interactive dashboard visualizations.

The project follows a complete data analytics workflow starting from data exploration in Google Colab, SQL-based data analysis in SQL Server, and dashboard creation in Power BI.

## Project Workflow

Customer_Shopping_Behavior_Analysis.ipynb notebook

This notebook contains:
- Data Import
- Data Exploration
- Data Cleaning
- Data Analysis using Python

##Connect the dataset to SQL Server
- Create database in SQL Server
- Load data from Python notebook into SQL Server
  
customer_behavior_sql_queries.sql
SQL Server Tasks:
- Created and used SQL database
- Imported customer shopping dataset
- Performed data analysis using SQL queries
- Generated customer and revenue insights using aggregations, CTEs, and window functions
  

##Connect SQL Server database to Power BI
customer_behavior_dashboard.pbix
Power BI Tasks:
- Connected SQL Server data to Power BI

- Built interactive dashboard visualizations
- Analyzed customer trends, revenue, and product performance
